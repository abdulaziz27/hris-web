# HRIS Logo Documentation

## Logo Specifications

### Logo Concept

HRIS logo menggunakan desain sederhana dan modern untuk sistem informasi sumber daya manusia.

### Color Palette

-   **Primary Blue**: #3B82F6 (Light mode) / #60A5FA (Dark mode)
-   **Text Dark**: #1F2937
-   **Text Light**: #F9FAFB
-   **Accent Colors**: Green (#10B981), Yellow (#F59E0B), Red (#EF4444)

### Logo Files

1. **logo-hris.svg** - Logo untuk light mode
2. **logo-hris-dark.svg** - Logo untuk dark mode
3. **favicon.svg** - Favicon sederhana

### Usage

-   Logo digunakan di Filament admin panel
-   Favicon ditampilkan di browser tab
-   Responsive dan scalable (SVG format)
-   Support dark/light mode

### Brand Name

**HRIS** - Human Resources Information System

### Implementation

Logo telah diintegrasikan ke:

-   ✅ Filament Admin Panel branding
-   ✅ HTML layouts (app.blade.php, auth.blade.php)
-   ✅ Favicon
-   ✅ App configuration (config/app.php, .env)
-   ✅ Composer.json metadata
